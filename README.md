# Livox_Mapping

Lidar Mapping using Lidar, GPS and IMU

using LVX SDK at <https://github.com/Livox-SDK/Livox-SDK/commit/9306596a2bf15c1343bc023b497465ed0a32909d>.

How to compile:

``` bash
cd build
cmake ..
make
```

Run binaries:

``` bash
cd build
./lvx_recorder
```

``` bash
cd build
./lvx_to_las
```
